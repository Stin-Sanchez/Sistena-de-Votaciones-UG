#  Sistema de Votaciones UG (Voting System)

Un sistema de votaciones sencillo y eficiente desarrollado en **C#**. Este proyecto permite gestionar un proceso electoral básico, desde la elección de candidatos hasta el conteo final de votos.

##  Descripción

El objetivo de este proyecto es simular un entorno de votación digital. Permite a los usuarios seleccionar una opción entre varios candidatos predefinidos y, al finalizar, el sistema calcula y muestra los resultados porcentuales y totales de la elección.

Es un proyecto ideal para demostrar fundamentos de programación en C#, lógica de control y manejo de estructuras de datos.

##  Funcionalidades Principales

* **Menú Interactivo:** Navegación sencilla para el usuario.
* **Gestión de Votos:** Registro seguro de la elección del usuario.
* **Validación de Datos:** Evita entradas erróneas o votos nulos no intencionados.
* **Conteo Automático:** Cálculo inmediato de los resultados al cerrar la votación.
* **Visualización de Resultados:** Muestra quién ganó y la cantidad de votos por candidato.

##  Tecnologías Utilizadas

* **Lenguaje:** C#
* **Framework:** .NET (Compatible con .NET 6/7/8 o .NET Framework)
* **IDE Recomendado:** Visual Studio 2022 / VS Code

##  Requisitos Previos

Para ejecutar este proyecto necesitas:
1.  Tener instalado el **.NET SDK** en tu equipo.
2.  Un editor de código como Visual Studio o VS Code.
